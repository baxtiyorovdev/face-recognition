# Dahua VTO 2111D-P-S3 · Контроль доступа по лицу (Pure Python)

## Структура проекта

```
dahua_face_access/
├── main.py              ← Запустить это
├── gui.py               ← Tkinter-интерфейс
├── face_engine.py       ← Детекция + антиспуфинг + распознавание
├── door_controller.py   ← HTTP API Dahua
├── config.py            ← ВСЕ настройки здесь
├── requirements.txt
└── known_faces/         ← Фото «своих» (создать вручную)
    ├── ivan_petrov.jpg
    └── anna_sidorova.jpg
```

## Установка

### 1. Зависимости (Ubuntu/Debian)

```bash
sudo apt-get update
sudo apt-get install -y python3-pip python3-tk cmake libboost-all-dev
pip install -r requirements.txt
```

### 2. Зависимости (Windows)

1. Установите [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/)
   (нужен для dlib, который использует face-recognition)
2. `pip install -r requirements.txt`

## Настройка

### config.py — обязательно измените:

```python
DAHUA_IP       = "192.168.1.110"   # ← IP вашего домофона
DAHUA_USER     = "admin"
DAHUA_PASSWORD = "ваш_пароль"
```

IP домофона: в веб-интерфейсе Dahua → Setting → Network → TCP/IP

### Добавление лиц (known_faces/)

```
known_faces/
├── ivan_petrov.jpg       →  распознаётся как "Ivan Petrov"
├── anna_sidorova.png     →  распознаётся как "Anna Sidorova"
```

**Требования к фото:**
- Одно лицо на изображение
- Анфас, хорошее освещение
- Минимум 200×200 пикселей

## Логика работы

```
Кадр с RTSP-камеры
        │
        ▼
  Детекция лиц (HOG)
        │
  ┌─────┴──────┐
Нет лиц      Лицо найдено
  │                │
 Ждём              ▼
             Антиспуфинг
             ─────────────────────────────
             1. Laplacian variance < 80?
                → ❌ ФОТО/РИСУНОК — игнор
             2. EAR (моргание глаз)
                → подтверждает живость
             ─────────────────────────────
                      │ OK — живое лицо
                      ▼
               Распознавание
               face_distance() vs база
               ─────────────────────
               confidence < 75%?
               → ❌ Неизвестный — отказ
               ─────────────────────
                      │ OK
                      ▼
                Таймер 3 секунды
                Лицо удерживается?
                      │
              ┌───────┴────────┐
             Нет              Да (3с истекло)
              │                │
           Ждём         HTTP GET openDoor
                        Dahua API → Замок открыт
                        Лог записан
```

## Антиспуфинг

| Метод | Описание |
|-------|----------|
| **Laplacian variance** | Анализирует текстуру кожи. Фото/рисунок имеют другую резкость |
| **Eye Aspect Ratio** | Обнаруживает моргание глаз (живой человек моргает) |
| **Hold 3 сек** | Лицо должно оставаться стабильным 3 секунды |

Для ещё более строгого режима раскомментируйте в `face_engine.py`:
```python
is_live = tex_ok and state.blink_done
```

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| `Не удалось открыть поток` | Проверьте IP, логин, пароль, сеть |
| `dlib не устанавливается` | `sudo apt-get install cmake build-essential` |
| Лицо не распознаётся | Добавьте больше фото в `known_faces/`, улучшите освещение |
| Фото проходит антиспуфинг | Уменьшите `LIVENESS_LAPLACIAN_THRES` в config.py |
| Живой блокируется | Увеличьте `LIVENESS_LAPLACIAN_THRES` или отключите EAR |

## Telegram Bot — Уведомления

### Как подключить

#### Шаг 1 — Создать бота
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте токен вида `123456789:ABCdefGHI...`

#### Шаг 2 — Узнать chat_id
1. Напишите вашему боту любое сообщение
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. В ответе найдите `"chat":{"id": 123456789}` — это ваш chat_id

#### Шаг 3 — Прописать в config.py
```python
TG_BOT_TOKEN        = "123456789:ABCdefGHI..."
TG_CHAT_ID          = "123456789"
TG_COOLDOWN_SECONDS = 30     # антиспам: раз в 30 сек на одно лицо
```

### Что бот отправляет

| Событие | Фото | Сообщение |
|---------|------|-----------|
| Кто-то подошёл (известный) | ✅ | 👤 Имя у двери + уверенность |
| Неизвестный у двери | ✅ | 🚨 Неизвестный посетитель |
| Дверь открыта (авто) | ✅ | 🔓 Имя + уверенность |
| Дверь открыта вручную | — | 🔓 Ручное открытие |

### Что изменилось (changelog)

| Параметр | Было | Стало | Эффект |
|----------|------|-------|--------|
| `HOLD_SECONDS` | 3.0 сек | **1.5 сек** | Дверь открывается в 2× быстрее |
| `FRAME_SKIP_GRAB` | 3 кадра | **2 кадра** | Меньше задержка RTSP-потока |
| `FACE_CONFIDENCE_THRES` | 0.40 | **0.45** | Точнее распознавание, меньше ложных срабатываний |
| `telegram_bot.py` | — | **добавлен** | Уведомления с фото в Telegram |
